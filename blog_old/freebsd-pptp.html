FreeBSD10.1 搭建PPTP服务器

最近频繁更换服务器，各种重装，各种配置，VPN服务是不可或缺的，每次配置都需要 Google，索性翻译过来，还可以深入学习一下。

原文地址：http://www.netroby.com/view.php?id=3598

安装要求

  1. FreeBSD 10.1
  2. 基于KVM的VPS或独立服务器
  3. 能够使用ssh配置你的服务器
  4. 能够使用FreeBSD Ports系统编译安装软件
  5. 能够通过Google查找答案（还是Google好用些）

注意

使用你的IP地址替换 27.12.32.17

安装步骤

  1. 编译 mpd5 并选择使用 ng_ipacct 。

    cd /usr/ports/net/mpd5 make install clean
